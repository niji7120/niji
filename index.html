    📖 사용 설명서
    👤 1. 프로필
    상단 사진 영역을 탭하면 프로필 사진을 등록할 수 있어요
    이름·클럽·나이대·손잡이·구력·스타일·라켓·스트링·텐션·스트링교체일을 저장하세요
    각 항목을 탭하면 바로 수정되고 자동 저장됩니다
    상단 🌙/☀️ 버튼으로 다크/라이트 테마를 전환할 수 있어요
    🏸 2. 경기 기록
    앱을 열면 날짜가 오늘로 자동 입력됩니다. 과거 날짜로 바꿀 수도 있어요
    상단 토글로 단식 / 복식을 선택하세요. 복식이면 파트너 입력란이 추가됩니다
    날짜 · 상대 이름 · 결과(승/패)는 필수 입력 항목이에요
    스코어 · 타이브레이크 · 메모는 선택 사항입니다
    저장된 경기는 목록에서 ✏️ 수정 · 🗑️ 삭제 가능해요
    🏆 대회에서 입력한 스코어는 경기기록에 자동 반영됩니다
    📊 3. 통계
    상단에서 🏸 배드민턴 / 🎾 테니스 종목을 전환할 수 있어요
    연승·연패 배너 — 현재 연속 기록이 자동 표시됩니다
    개인 요약 — 통산 승률, 단식·복식 별도 승률 확인
    월별 승률 그래프 — 월별 성적 추이를 바 차트로 시각화
    상대 전적 · 파트너 궁합 · 상대팀 상성 분석
    👥 4. 선수 관리
    선수 등록 시 이름·나이대·손잡이·구력·플레이스타일을 함께 입력할 수 있어요
    선수 목록에서 이름을 탭하면 상세 전적 페이지로 이동합니다
    나와의 전적 (승·패·승률%), 최근 대전 기록을 확인하세요
    클럽 관리에서 클럽을 만들고 선수에게 배정할 수 있어요
    🏆 5. 한울방식
    파트너와 상대방이 매 게임 바뀌는 순환 복식 대회를 관리해요
    + 새 대회 만들기를 탭해 대회명·날짜·게임수를 설정하세요
    선수 추가 후 ↑↓ 버튼으로 순서를 조정하면 시드 배정이 가능해요
    대진표에서 스코어를 입력하면 경기기록에 자동 반영됩니다
    순위표에서 🥇🥈🥉 순위를 확인하세요
    💾 6. 백업 & 복원
    데이터는 클라우드(Firebase)에 자동 저장됩니다. 폰을 바꿔도 그대로 유지돼요
    데이터 내보내기 — 모든 기록을 JSON 파일로 저장해 추가 백업할 수 있어요
    가져오기 — 백업 파일을 불러와 데이터를 복원할 수 있어요
    🧹 저장공간 정리 — 폰 용량이 부족할 때 탭하세요. 클라우드 데이터는 안전하게 유지됩니다
    💬 7. 니지스트링 커뮤니티
    공유 탭에서 카카오 오픈채팅에 참여해 동호인들과 소통하세요
    스트링 교체, 라켓 수리 등 니지스트링 전문 서비스를 이용하세요
