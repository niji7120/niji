    📖 사용 설명서
    👤 1. 프로필
    상단 사진 영역을 탭하면 프로필 사진을 등록할 수 있어요
    라켓을 최대 4개까지 등록할 수 있어요. 각 라켓마다 스트링·텐션·교체일을 따로 관리해요
    스트링 교체일을 입력하면 상단 배너에 D+일수와 게임 수가 표시돼요
    라켓이 2개 이상이면 배너를 좌우로 스와이프해서 각 라켓 상태를 확인할 수 있어요
    상단 🌙/☀️ 버튼으로 다크/라이트 테마를 전환할 수 있어요
    🏸 2. 경기 기록
    앱을 열면 날짜가 오늘로 자동 입력됩니다
    상단 토글로 단식 / 복식을 선택하세요
    날짜 · 상대 이름 · 결과(승/패)는 필수 입력 항목이에요
    저장된 경기는 목록에서 ✏️ 수정 · 🗑️ 삭제 가능해요
    🎾 3. 스트링 관리
    상단 탭으로 라켓을 선택한 뒤 교체일을 입력하세요
    교체 이력을 추가하면 자동으로 최근 교체일이 갱신돼요
    교체 권장 주기가 지나면 🔴 빨간색 경고가 표시됩니다
    💾 4. 백업 & 복원
    데이터는 클라우드(Firebase)에 자동 저장됩니다
    데이터 내보내기 — 모든 기록을 JSON 파일로 저장해 추가 백업할 수 있어요
